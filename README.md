Необходимо сделать на фреймворке Yii1 + MySQL каталог книг. Книга может иметь несколько авторов. Тестовое задание можно делать без верстки. 

1. Книга - название, год выпуска, описание, isbn, фото главной страницы.
2. Авторы - ФИО.

Права на доступ: 
1. Гость - только просмотр + подписка на новые книги автора. 
2. Юзер - просмотр, добавление, редактирование, удаление.
 
Отчет - ТОП 10 авторов, выпустившие больше книг за какой-то год (доступен всем пользователям). 

ПЛЮСОМ БУДЕТ
Уведомление о поступлении книг из подписки должно отправляться на смс гостю.

https://smspilot.ru/


Для запуска потребуется docker
Необходимо выполнить несколько команд
1. Запустить контейнеры docker compose up -d
2. Выполнить команду: docker compose exec php_fpm composer install
